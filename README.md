
# proyecto_ds

Proyecto inicial de **Ciencia de Datos** enfocado en la exploración básica de datos utilizando Python y herramientas del ecosistema de análisis como **Pandas**, **NumPy**, **Jupyter Notebook** y **Streamlit**.

## Estructura del Proyecto

```
proyecto_ds
│
├── .venv/
├── requirements.txt
├── README.md
│
└── notebooks/
    └── 01_exploracion_inicial.ipynb
```

## Tecnologías Utilizadas

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Streamlit

## Instalación

### 1. Crear el entorno virtual

python -m venv .venv

### 2. Activar entorno

Windows (PowerShell)

.\.venv\Scripts\Activate.ps1

macOS / Linux

source .venv/bin/activate

### 3. Instalar dependencias

pip install -r requirements.txt

## Uso

Iniciar Jupyter Notebook:

jupyter notebook

Luego abrir:

notebooks/01_exploracion_inicial.ipynb

## Verificación

python -c "import pandas, numpy, streamlit; print('OK')"

Si todo está correcto se mostrará:

OK

Este proyecto establece una base para:
